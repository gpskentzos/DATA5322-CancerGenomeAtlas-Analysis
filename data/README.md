# Data

The dataset is too large to store in this repository. Download it before running any notebooks.

## Download Instructions

1. Go to the Kaggle dataset page:
   **https://www.kaggle.com/datasets/orvile/gene-expression-profiles-of-breast-cancer**

2. Download and unzip into this `data/` directory. The final structure should look like:

```
data/
├── BC-TCGA/         # TCGA tumor + normal gene expression (primary dataset)
├── GSE2034/         # GEO Series GSE2034 tumor + normal samples
├── GSE25066/        # GEO Series GSE25066 tumor + normal samples
└── Simulation-Data/ # Simulated data for testing
```

## Official Source

The original data comes from The Cancer Genome Atlas (TCGA):
https://portal.gdc.cancer.gov
